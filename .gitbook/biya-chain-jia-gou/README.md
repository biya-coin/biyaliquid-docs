# Biya Chain 架构

## 什么是模块?​

模块是Injective区块链架构中的基础组件，每个模块都旨在提供特定的功能。一个模块本质上是一个具有明确定义的逻辑和服务的自包含单元，使Injective能够高效地管理网络上的各种操作。

模块就像积木一样，可以组合起来以扩展区块链的功能。这种方法减少了实现核心功能的摩擦和技术复杂性，使开发更加容易。每个模块独立运行，但通过模块间的消息传递与其他模块通信，使Injective的架构具有模块化和可适应性。这使得Injective能够持续演进，提供有针对性的升级和功能，而无需重新构建整个系统。
