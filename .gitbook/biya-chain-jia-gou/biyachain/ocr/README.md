# OCR

## 摘要

OCR 模块用于将 Chainlink 的 OCR（链下报告）信息存储到链上存储中。

Feed 配置由模块管理员管理，报告由传输者和观察者提交到链上。传输者和观察者通过治理配置的链上 LINK 代币获得奖励。
