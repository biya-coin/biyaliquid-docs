# 错误代码

本文档列出了模块中使用的错误代码。


| 模块        | 错误代码 | 描述                                     |
|-------------|----------|------------------------------------------|
| tokenfactory | 2        | 尝试创建已存在的代币单位（已有银行元数据） |
| tokenfactory | 3        | 未授权的账户                             |
| tokenfactory | 4        | 无效的代币单位                           |
| tokenfactory | 5        | 无效的创建者                             |
| tokenfactory | 6        | 无效的权限元数据                         |
| tokenfactory | 7        | 无效的创世状态                           |
| tokenfactory | 8        | 子代币单位太长，最大长度为 %d 字节       |
| tokenfactory | 9        | 子代币单位太短，最小长度为 %d 字节       |
| tokenfactory | 10       | 嵌套子代币单位太短，每个应至少 %d 字节   |
| tokenfactory | 11       | 创建者太长，最大长度为 %d 字节           |
| tokenfactory | 12       | 代币单位不存在                           |
| tokenfactory | 13       | 金额必须为正数                           |
