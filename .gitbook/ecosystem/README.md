# 生态入口

### 官方网站

Biya Chain 官方网站，了解项目愿景、技术特性和最新动态。

### 区块链浏览器

实时查看区块链上的交易、区块、地址和智能合约等信息。

### Biya Chain Hub

一站式管理您的数字资产、参与质押和治理。

主要功能:

* **钱包管理**: 创建和管理您的 Biya Chain 钱包
* **质押**: 质押 BIYA 代币参与网络安全并获得奖励
* **治理**: 参与链上治理投票，决定协议的发展方向

### 公链管理台

Biya Chain 公链管理平台，为验证者和节点运营商提供专业的链管理工具。

主要功能:

* **节点管理**: 监控和管理您的验证者节点
* **网络状态**: 实时查看网络健康状况和性能指标
* **系统配置**: 配置和优化节点参数
* **日志查看**: 查看节点运行日志和错误信息
