## Developer Resources

### Developer Tools and Resources

Developer tools and resources to get you building on Biyaliquid

| **Resource**                                                                                  | **Description**                                                                                                             |
| --------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| [Biyaliquid 101](https://biyaliquid.notion.site/biyaliquid-101-589dedc4c9c04531aae503dbb235d443) | One-stop-shop for Biyaliquid resources                                                                                       |
| [Biyaliquidd](./developers/biyaliquidd/README.md)                                                            | Command-line interface and node daemon that connects to Biyaliquid                                                           |
| [Biyaliquid Explorer](https://prv.scan.biya.io/zh/transactions/)                                     | Analytics platform that enables anyone to search addresses, trades, tokens, transactions, and other activities on Biyaliquid |
| [Biyaliquid Local](https://github.com/biya-coin/biyaliquid-local)                           | Biyaliquid testnet and ecosystem containerized with Docker and orchestrated with a simple docker-compose file.               |
| [Biyaliquid REST API](https://lcd.biyaliquid.network/swagger/)                                  | Swagger API explorer                                                                                                        |
| [Biyaliquid TypeScript SDK](https://docs.ts.biyaliquid.network)                                 | Build dApps on Biyaliquid using TypeScript                                                                                   |
| [Biyaliquid API Reference](https://api.biyaliquid.exchange)                                     | Detailed API documentation for interacting with Biyaliquid for traders                                                       |
| [Cosmovisor](./infra/cosmovisor.md)                                                 | Small process manager around Cosmos SDK binaries that monitors the governance module                                        |
| [CosmosSDK](https://docs.cosmos.network/main/build)                                          | The Cosmos SDK documentation serving as a valuable resource for developers integrating with the Biyaliquid ecosystem         |

### Helpful Repos

Helpful repositories for developing on and integrating Biyaliquid

| **Repo**                                                                                         | **Description**                                                                                                                      |
| ------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------ |
