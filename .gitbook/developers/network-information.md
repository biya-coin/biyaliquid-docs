# Network Information

## Biyaliquid Testnet Details

* Chain ID: `biyaliquid-888`
* Node: `https://testnet.sentry.tm.biyaliquid.network:443`

{% hint style="info" %}
Note that the Biyaliquid Chain ID for EVM is `1439`.
However, it natively uses a chain ID of `biyaliquid-888`.
While the chain IDs are different, they map to the **same** network.

See [EVM network information](../developers-evm/network-information.md) for more details.
{% endhint %}
