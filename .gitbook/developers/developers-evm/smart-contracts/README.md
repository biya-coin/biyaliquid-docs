# 您的第一个 EVM 智能合约

智能合约是在区块链上运行的代码。您可以编译 Solidity 智能合约，并在 Biya Chain 的以太坊虚拟机 (EVM) 上测试、部署、验证和与之交互。您也可以使用标准的 EVM 开发工具和框架来完成这些操作。
这些指南将向您展示如何使用 Hardhat 和 Foundry 来实现。

## Hardhat

本指南将引导您使用 [Hardhat](https://hardhat.org/) 在 Biya Chain 测试网上构建 EVM 智能合约。

* [设置 Hardhat 并编译智能合约](./compile-hardhat.md)
* [使用 Hardhat 测试智能合约](./test-hardhat.md)
* [使用 Hardhat 部署智能合约](./deploy-hardhat.md)
* [使用 Hardhat 验证智能合约](./verify-hardhat.md)
* [使用 Hardhat 与智能合约交互](./interact-hardhat.md)
