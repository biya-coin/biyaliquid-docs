# 关于 Biya Chain

## 关于 Biya Chain

### 什么是 Biya Chain？

Biya Chain 是为金融而生的区块链。

它是唯一一个开发者可以使用预构建、可定制的[模块](developers/developers-native/)来创建其他网络上无法实现的动态应用的区块链。结合其核心架构的优化和增强的跨链互操作性，Biya Chain 提供了一个高性能网络，能够高效、安全地将全球金融系统上链。
