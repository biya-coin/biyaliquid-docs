# 关于 Biyaliquid

## 什么是 Biyaliquid？

Biyaliquid 是为金融而生的区块链。

它是唯一一个开发者可以使用预构建、可定制的[模块](developers-native/README.md)来创建其他网络上无法实现的动态应用的区块链。结合其核心架构的优化和增强的跨链互操作性，Biyaliquid 提供了一个高性能网络，能够高效、安全地将全球金融系统上链。
