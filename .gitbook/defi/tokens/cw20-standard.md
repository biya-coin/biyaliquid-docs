# CW20 标准

CW20 代币标准提供了一个用于无需许可创建和管理可替代代币的框架，更类似于 [ERC20 标准](https://ethereum.org/en/developers/docs/standards/tokens/erc-20/)。如上所述，由于 TokenFactory 与 Cosmos SDK 的原生集成，鼓励使用 TokenFactory，但如果您出于任何原因希望使用 CW20 标准，您可以使用 [CW20 适配器](https://github.com/CosmWasm/cw-plus/blob/main/packages/cw20/README.md)将 CW20 代币转换为 TokenFactory 代币，反之亦然。有关 CW20 标准的更多信息，请参阅其正式规范[此处](https://github.com/CosmWasm/cw-plus/blob/main/packages/cw20/README.md)。
