# Command Line Node Interaction

You can use the `injectived` CLI to interact with a node. If you are interacting with a node in your local private network, make sure the node is running in the terminal before you use the CLI.

For more details on how to use `injectived`, see [https://github.com/InjectiveLabs/injective-docs/blob/master/.gitbook/developers/injectived/use.md](https://github.com/InjectiveLabs/injective-docs/blob/master/.gitbook/developers/injectived/use.md "mention").
