# Command Line Node Interaction

You can use the `biyaliquidd` CLI to interact with a node. If you are interacting with a node in your local private network, make sure the node is running in the terminal before you use the CLI.

For more details on how to use `biyaliquidd`, see [https://github.com/biya-coin/biyaliquid-docs/blob/master/.gitbook/developers/biyaliquidd/use.md](https://github.com/biya-coin/biyaliquid-docs/blob/master/.gitbook/developers/biyaliquidd/use.md "mention").
