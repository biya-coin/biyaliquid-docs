# 关于 Biyachain

## 什么是 Biyachain？[​](https://docs.injective.network/learn/introduction#what-is-injective)

Biyachain 是专为金融而构建的区块链。

Biyachain 是一个允许开发者使用预构建且可定制模块 [modules](kai-fa-zhe/mo-kuai/) 来创建其他网络上无法实现的动态应用程序的区块链。结合对其核心架构的优化和增强的跨链互操作性，Biyachain 提供了一个高性能网络，准备高效且安全地将全球金融系统引入链上
