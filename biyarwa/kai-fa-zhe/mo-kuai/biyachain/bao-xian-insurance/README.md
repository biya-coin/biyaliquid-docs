# 保险(Insurance)

## 摘要

本文规定了 Biyachain 链的保险模块。\
该模块为 Biyachain 链的交易模块中的衍生品市场提供保险资金，以支持更高杠杆交易。从高层次来看，每个衍生品市场的保险资金由一个无许可的承保人群体提供资助，每个承保人拥有对保险基金中基础资产的按比例索赔权（通过保险基金份额代币表示）。\
当其对应的衍生品市场中的头寸被强平并且有正的权益时，保险基金会增长，因为在强平时，正的权益的一半将被发送到保险基金。当一个负权益的头寸被强平时（即头寸已超过破产），保险基金将被用于覆盖缺失的权益。
